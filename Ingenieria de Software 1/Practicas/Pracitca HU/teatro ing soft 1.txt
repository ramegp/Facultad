ID Reserva de Entrada

Esc 1 Reserva Exitosa
	Dado: Un dni y nombre, una funcion con cupo disponible, con una anticipacion mayor a 3 horas, y la cantidad sea menor a 3
	Cuando: El empleado carga los datos
	Entonces: El sistema resta la cantidad de entradas y reserva las mismas
Esc 2 Reserva mayor a 2
	Dado: un dni xxxx que quiere reservar mas de 2 entradas
	Cuando:El empleado registra los datos
	Entonces: El sistema informa que supera la cantidad permitida
Esc 3:No hay cupo
	Dada: una funcion seleccionada sin cupo
	Cuando: El empleado intenta reservar para esa funcion
	Entonces: El sistema informa "no hay cupo disponible"
Esc 4: Anticipacion menor 3hs
	Dada: una funcion seleccionada con inicio dentro de <3hs
	Cunado: El empleado intenta reservar para esa funcion
	Entonces: El sistema informa que no es posible la reserva por inicio proximo
